Computer-aided Generation of the Thermodynamic Properties of R134a

By Nwankwo Daniel

Abstract

A computer package for the generation of the thermodynamic properties of Refrigerant 1, 1, 1, 2-tertrafloroethane (R134a) an alternative refrigerant to R12, which has no Ozone Depletion Potential has been developed based on an appropriate equations of state. The equations were used to develop a software in the MATLAB graphic user interface development environment. The software is able to compute the thermodynamic properties of R134a in the single (subcooled and superheated), saturated and wet vapour regions. The calculated values are accurate compared to the standard reference property tables for R134a.
The software is easy to install and use, does not require internet as opposed to the available sophisticated and expensive computer software packages. The software would be useful to both practicing engineers and engineering students in refrigeration area. The software can equally be used to generate thermodynamic property tables which are automatically opened in Excel worksheet. An application was illustrated by generating tables in the three sections of the software. Equally a typical refrigeration plant problem was solved in Excel worksheet using the values generated by the software.

Keywords: MATLAB Applications, thermodynamic properties of R134a refrigerants Excel add-in

Introduction

1, 1, 1, 2-tertrafloroethane, CF3CH2F or R134a is an environmentally friendly refrigerant which has replaced the ozone depleting R12 (dichloro-difluoro-methane) in a wide range of application especially in automotive air conditioning and domestic refrigeration (Huber and Ely, 1992). Because of its current and future importance, accurate knowledge of its thermodynamic properties is essential (Reiner and Hans, 1994). Tables and charts of the thermodynamic properties of R134a have been existing as working tools for both students and practicing engineers but the need for a computer software that automatically generates the values of the thermodynamic properties keeps arising as it is strenuous, time consuming and error prone to read up values from the tables and charts of properties.
A fundamental equation for the Helmholtz free energy of R134a, valid for temperatures between 170 K and 455 K and pressures of up to 70 MPa which has been compared to equations established by other research groups by Annex 18 of the International Energy Agency (IEA) and has been selected as an international standard formulation for the thermodynamic properties of R134a was presented by Reiner and Hans (1994).
Computer technology has made solutions to engineering problems fast and accurate thereby making the acquisition of computer skills a priority for the contemporary engineer. In order to meet up with demanding and competitions among engineering system the engineers seek solutions to problems of the engineering systems developed or to be developed with view of fastness and accuracy. Thus, the contemporary engineer that cannot proffer solutions to most real life problems with computer systems may not survive in this competitive engineering world. The Engineer solves the engineering problems by formulating sequence of procedures and that when carefully followed it gives accurate result at all times. This trend gives rise to the development of computer software for solving engineering problems, as when required. In the early years of machine languages, developing computer software or package was very tedious and time consuming because a piece of software may consist of millions of lines of code. But in our modern day, due to the emergence of object/procedure-oriented programming languages (e.g. Visual Basic, Visual C++, Visual Pascal, MatLab etc), the complexity of generating or writing millions of code of lines has reduced tremendously (Beckman and Klein, 1996).

Thermodynamic properties of refrigerants are important in refrigerating process analysis and system design. The property data are usually determined semi-empirically and are either tabulated or provided as charts of properties. But, the use of property tables frequently requires interpolation between the tabulated data, which is time consuming and reading the property charts is also strenuous, time consuming and always prone to errors. However, the extensive use of computing facilities in contemporary engineering practice limits the use of property charts (Dromey, 1985). The trend is towards the development of computer packages that are capable of automatic generation of the values of the desired properties, and thus, facilitating their use in engineering analysis. The use of thermodynamics properties computer packages can improve the curiosity of engineering students in the understanding of the properties tables and charts (Tan and Chua, 2003).
Beckman and Kelvin (1996) developed a commercial software package called Engineering Equation Solver (EES) that has the capability of analyzing thermodynamic and transport properties of many fluids used in engineering practises. Refprop is FORTRAN based software that allows the user to look up thermodynamic properties for most refrigerants currently in use.(Lemon and others, 2002). Maurizio and Sonia (2004) developed software for the analysis of thermodynamic properties of refrigerant R134a using Visual C++ programming language. Oko and Diemodike (2006) developed an Excel-add in Thermoprop-R152a for generating the thermodynamic properties of refrigerant R152a.

MATLAB is an acronym for matrix laboratory. It integrates computation visualization and programming environment thereby making it a high performance language for technical computing. MATLAB can be used as a software for carrying out mathematical calculations (mathematical computations, modelling, simulation data analysis, graphics etc) and as a programming language for developing computer programs (Tamuno, 2011). The present of graphic user interface development environment (guide) in MATLAB makes it an object oriented programming language. MATLAB GUI makes it possible the development of user friendly and interactive platforms. The use of guide GUI forms the basis of this work.

Most of the computer software packages for the determination of thermodynamic properties and simulation of refrigeration processes require subscription, which are generally expensive for the developing countries. One way to make some of these computer tools available to the developing countries is through the Internet. With the rapid evolution of Internet, there are now several websites that provide online computation of thermodynamic properties for pure substances. There is a good agreement between the results generated by the free online Internet applets and those generated by the commercial desktop application software (Tan and Chua, 2003). However, a report has shown that about 13.5% only of African population has Internet access (Internet World Stats, 2012), therefore, Internet applets for thermodynamic properties may not be available to the majority of African students.

Governing Equations

The equation of state for R134a presented here is a fundamental equation of state for the dimensional Helmholtz free energy (Reiner and Hans, 1994).

Results and Discussion

Using the Software to Generate Property Values

Before installing the software, it is necessary to install MATLAB Software.Once the software is installed, a new GUI menu is available on the main MATLAB menu which can be called up by typing R134a2 in the MATLAB command window. The software is divided into three sections; the single phase section which encompasses the superheated and the subcooled phases, the saturated phase section, and the wet phase section. The top of the software for each section accepts the user input data, and the results are shown at the down part. The units of both the input and output parameters are appended to the properties by design.
. Figure 1 shows the outcome of using the software to generate properties in the three sections. In the single phase section, temperature and pressure are selected with values of 0 (oC) and 10 (kPa), respectively, temperature of value, 20 (oC) is selected in the saturated phase section while quality fraction and saturation temperature of values, 0.7 (-) and 10 (oC), respectively are selected in the wet phase section. It is easily discovered that the wet phase section is divided into two subsections. One subsection calculates the latent properties using quality fraction and saturation properties as the input parameters, while the other subsection calculates the quality fraction using the latent and the saturation properties as the input parameters.
The saturation temperature and pressure of the selected state are automatically calculated and compared with the temperature and pressure of the state to determine whether it is in the subcooled or superheated phase. The values of saturation temperature and pressure of the selected state are shown on the Sat. properties panel while the phase of the state is shown on the Status row.





The software has button in each section captioned Table used for generating thermodynamic property tables. Clicking the button brings forth a dialog box that collects data from the user for the property table. After selecting parameters and setting constants and limits for the table as done in Figures 2, 4, and 6, the corresponding tables are generated when the view table button is clicked and opened in Microsoft Excel worksheet as sown in Figures 3, 5, and 7. The Close button in each of the dialog boxes is used to close down the dialog box.

Conclusion

MATLAB software is cheap and always available in almost all software shops. To acquire and install it in a system is easy, and the use of the software for thermodynamic properties developed using MATLAB is cheaper when compared with the Internet applets especially when African countries are into consideration where internet access is limited. The present work presents computer software developed in MATLAB graphic user interface development environment. The software is able to compute the thermodynamic properties of R134a refrigerant - specific volume, internal energy, enthalpy, entropy, isobaric heat capacity and isochoric heat capacity in the states of saturated, compressed liquid, and superheated vapour. Also, the software can be used to generate thermodynamics table which are automatically opened in Microsoft Excel worksheet. The values generated by the software is used to solve refrigerating plant problem using Excel worksheet The software is a veritable tool for the effective teaching of mechanical engineering students how to design and simulate vapour compression refrigeration systems. The solution scheme and computer language adopted in this work are easy to apply and use as opposed to the sophisticated and expensive computer software used by most researchers in the area. The accuracy of its output is good enough for engineering practice and, therefore, strongly recommended as a teaching tool, which would be useful in regions where there is limited internet access.

Recommendation

The tool proposed in this work is easy to install, use and modify by engineering students in any computer. It is, therefore, strongly recommended as a teaching tool for engineering students (for example, in the thermal design, performance analysis and simulation of refrigeration). This work should be joined with the software that generates thermodynamic properties of R152a developed by Oko and Diemuodeke (2010). More software should equally be developed for the thermodynamic properties of other stratospherically safe refrigerants.
The computer interactive Problem Solver Developed by Mezie-Okoye (2004) has some very good features to solve thermofluid problems but it requires the user to supply input data manually, which is, in most cases, time consuming.
